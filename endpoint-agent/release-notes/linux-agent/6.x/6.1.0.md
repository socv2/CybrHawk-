# 6.1.0

#### Version 6.1.0

Release Date: 2024-12-13

**What's New**

* New install argument `--auto_update` (Default value = `Yes`). Declaring this variable as `No` will disable automatic updates for TD\_Endpoint.
* New `--proxy` option to configure explicit http proxy settings for agent communication with TD servers.

**Bug Fixes**

* Service performance optimizations and stability fixes.
* Automatic upgrade bug fixes.
