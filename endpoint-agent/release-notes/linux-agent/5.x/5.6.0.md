# 5.6.0

#### Version 5.6.0

Release Date: 2023-09-11

**What's New**

* The agent now communicates to the ThreatDefence SecOps platform exclusively over TCP port 443.
* Streamlined activation process: a single activation code is used instead of configuration files.
* Background self-updating improved: agents can automatically update to the latest version.
* Velociraptor endpoint integrated: Providing deeper forensic and incident response ability to your endpoints.

**Bug Fixes**

* Intermittent connection dropout issues resolved.
* Service performance optimizations and stability fixes.
