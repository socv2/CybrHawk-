# Prerequisites

## Network and System Requirements

The CybrHawk agent requires outbound connectivity to several hosts over **TCP port 443**. These connectivity requirements are detailed in the **Customer Portal**.

***

## Supported Platforms

* **Processor**: x86\_64 processor and operating system.

***

## Supported Operating Systems

#### Windows

* Windows 11
* Windows 10
* Windows Server 2016
* Windows Server 2019
* Windows Server 2022
* Windows Server 2025.

#### Linux

* Debian 11 or later
* Ubuntu 18.04 or later
* Red Hat Enterprise Linux (RHEL) 7 or later
* Oracle Linux 7 or later
* Amazon Linux 2 or later
* SUSE Linux Enterprise Server (SLES) 15.3 or later.

#### MacOS

* macOS 13 (Ventura) or later.
