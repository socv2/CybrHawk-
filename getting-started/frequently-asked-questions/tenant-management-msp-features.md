# Tenant Management

***

## Is the platform multi-tenant?

Yes. The platform is fully multi-tenant, with all tools supporting multiple tenants and providing strict data segregation between them.

## Do we need to use different web interfaces for each tenant?

No. All tenant data is visible in the same UI. By default, you can view all tenants together and drill down into any specific tenant as needed.

## Can I grant access so one of my team members can only access a subset of tenants?

Yes. Granular permissions can be applied to restrict access so that users can only see a defined subset of tenants.
