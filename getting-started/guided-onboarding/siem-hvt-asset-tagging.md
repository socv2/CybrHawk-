# SIEM HVT Asset Tagging

High-Value Target (HVT) assets are those whose compromise could result in a severe business impact, such as operational downtime, data breach, or financial loss. Examples include key personnel like the CFO or system administrators, critical servers, and devices with privileged access.

Tagging these assets enables you to proactively search for and generate alerts on all related activity. You can easily isolate these critical events in your SIEM using a query like `tags:hvt*` alongside other relevant tags.
