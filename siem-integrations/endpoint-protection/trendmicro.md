# Trend Micro

To integrate Trend Micro with CybrHawk, follow these steps:

## Step 1: [Activating Log Forwarder API in Worry-Free Business Security Services (WFBS-SVC)](https://success.trendmicro.com/dcx/s/solution/1118040-activating-log-forwarder-api-in-worry-free-business-security-services-wfbs-svc)

1. Send a request for access to the Log Forwarder API to Trend Micro's WFBS-SVC Technical Support team. Send your request along with your WFBS-SVC Activation Code/s by contacting [Trend Micro Technical Support](https://success.trendmicro.com/contact-support).
2. The WFBS-SVC Technical Support team will send you the Cloud Services Platform Integration (CSPI) key pair, which is required to setup Log Forwarder.

## Step 2: Configuration in CybrHawk

**Provide CybrHawk with Client Information**:

* Provide the following information to your CybrHawk representative at [CybrHawk Support](mailto:socv2@cybrhawk.com):
* Trend Micro API Key
* Trend Micro API Secret.
