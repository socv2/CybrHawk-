# Software as a Service

The ThreatDefence platform supports integrations with popular **SaaS (Software-as-a-Service)** platforms.\
These integrations provide visibility into user activity, access patterns, audit logs, and security events from third-party applications.\
This section contains step-by-step integration guides for the most common SaaS vendors.
