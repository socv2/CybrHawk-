# JumpCloud

By integrating **JumpCloud** with **CybrHawk** via JumpCloud’s API and event logging capabilities, you can stream identity, access, and device-related events directly into CybrHawk for comprehensive visibility and centralized security monitoring.

This integration enables CybrHawk to track:

* User authentication
* Directory changes
* Policy enforcement
* Endpoint activity

By correlating this with other security data, CybrHawk supports Zero Trust access control, enhances threat detection, and accelerates incident response.

***

## Step 1. Generate API Key

1. On your JumpCloud dashboard, click your **account initials** in the top-right corner.
2. Navigate to **My API Key**.
3. Select **Generate New API Key** and store the value in a secure location.

***

## Step 2. Configure CybrHawk Integration

Provide the following information to your CybrHawk representative at\
📧 [**CybrHawk Support**](mailto:support@threatdefence.com):

* API Key
