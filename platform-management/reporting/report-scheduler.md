# Automated Report Scheduler

Reports can be scheduled to be emailed at regular intervals. To access the report scheduler, please navigate to Customer Portal/My Tenants/Report Scheduling:

* Set the recipient email address in the "To:" section. It is recommended that this be a security distribution group, or you can specify multiple recipients.
* Set the email interval by selecting report names to be emailed under the Weekly or Monthly section and click confirm.
* To disable all report emails, click reset to default or delete the recipients.
